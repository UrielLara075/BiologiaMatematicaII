# Mathematical Modeling Projects

Collection of mathematical modeling projects developed for the Mathematical Biology II course. This repository includes simulations, data analysis and reports focused on biological systems.

## Projects

### Wildfire Propagation Model

Simulation of wildfire spread under different enviromental conditions on a grid using an agent-based model in NetLogo. The project investigates the influence of enviromental parameters on propagation dynamics through simulated experiments and exploratory daya analysis in python.

#### Files:
- Report: Wildfire_propagation_report.pdf
- Python script: threshold.ipynb
- Visual outpot: output_v1.png to v_3
- Various iterations of the simulation: Fire_v1.nlogo to v3
- Base model: Fire.nlogo
- [Simulation results](https://drive.google.com/drive/folders/1lMz77mPqmsNFcQdjFWOKNuf6jusvXEqF?usp=sharing) (not stored in this repository)

### Cicada Synchronization
Implementation and analysis of a mathematical model describing reproduction cycles of cicadas and exploring the periodic synchronization phenomenon.

#### Files:
- Report: Cidada_synchronization_report.pdf
- Multiple interpretation of the model present in literature: cicada_v1.ipynb to v3

### Ant trail selection
Adaptation of a continous model from the literature into a discrete agent-based simulation of ant shortest path selection through pheromone trails.

#### Files:
- Report: ant_trail_selection_report.pdf
- Simulation: AntsNetwork.nlogo

# Mathematical Modeling Projects

Collection of mathematical modeling projects developed for the Mathematical Biology II course. This repository includes simulations, data analysis, and reports focused on biological systems.

## Projects

### Wildfire Propagation Model

Simulation of wildfire spread under different environmental conditions on a grid using an agent-based model implemented in NetLogo. The project investigates the influence of environmental parameters on propagation dynamics through simulation experiments and exploratory data analysis in Python.

**Files:**

- Report: `Wildfire_Propagation_Report.pdf`
- Python script: `threshold.ipynb`
- Visual output: `output_v1.png` to `output_v_3.png`
- Various iterations of the simulation: ``Fire_v1.nlogo`` to `Fire_v3.nlogo`
- Base model: `Fire.nlogo`
- [Simulation results](https://drive.google.com/drive/folders/1lMz77mPqmsNFcQdjFWOKNuf6jusvXEqF?usp=sharing) (stored externally)

### Cicada Synchronization

Implementation and analysis of a mathematical model describing reproduction cycles of cicadas and the emergence of periodic synchronization.

**Files:**

- Report: ``Cicada_Synchronization_Report.pdf``
- Alternative implementations of the model: ``cicada_v1.ipynb`` to ``cjcada_v3.ipynb``

### Ant Trail Selection

Adaptation of a continuous model from the literature into a discrete agent-based simulation of ant path selection through pheromone trails.

**Files:**

- Report: ``Ant_Trail_Selection_Report.pdf``
- Simulation: `AntsNetwork.nlogo`
